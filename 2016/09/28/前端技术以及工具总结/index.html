前端技术以及工具总结

  • 前端技术

最近一段时间由于工作需要开始接触前端技术，在此总结下这段时间新接触的很多技术（包括语言、工具以及框架）。

JavaScript

大家都知道js非常火，从前端到后端的js如今已经俨然成为全能王。刚开始学习自己也是被原型链继承和动态语言的各种独特语法搞蒙圈。网上的优秀的教程实在太多，在此做个搬运工，推荐一下入门JavaScript资料和网站。

  1. 《JavaScript高级程序设计》很经典，网上很容易下载到电子版
  2. MDN上关于JavaScript的讲解，MDN的web技术文档简直太强大和全面，绝对值得收藏。https://developer.mozilla.org/zh-CN/docs/Web/JavaScript
  3. 阮一峰关于es6的讲解 http://es6.ruanyifeng.com/#docs/class
  4. 在线运行JavaScript的好工具 https://jsfiddle.net/

Typescript

作为之前编写静态语言（C语言、java）的程序员，一定会为JavaScript的动态语法特点烦恼，而Typescript就是伟大的软微提出的一套作为JavaScript超集的Typescript语言。最大的不同就是带有静态类型检测，易于构建更加健壮的代码。

  1. Typescript中文官方文档 http://tslang.cn/docs/tutorial.html

npm、cnpm、nvm

npm是基于nodejs的JavaScript第三方库的管理工具，类似于ubuntu下的apt-get、ruby下的gem、mac下的homebrew。但是因为npm下载第三方库的源在国外，通常我们用cnpm（npm淘宝的镜像）来install我们需要的JavaScript库。而node发展之快导致很多我们自己开发的代码兼容性有问题，nvm就是管理电脑上node版本的开源工具，可以随意安装并切换所有node的历史版本。

webpack、gulp

webpack和gulp都是类似的web项目进行打包压缩等一系列处理的开源工具。通常大型web项目都需要依赖webpack或gulp或两者结合或者其他的工具（例如grunt）来完成对所有前端代码文件资源的自动化处理。
1.webpack快速入门 http://www.cnblogs.com/vajoy/p/4650467.html
2.webpack官网教程 http://webpack.github.io/docs/tutorials/getting-started/
3.gulp快速入门 http://w3ctrain.com/2015/12/22/gulp-for-beginners/?utm_source=tuicool&utm_medium=referral
4.gulp官网教程 http://www.gulpjs.com.cn/docs/

React、flux

React无疑这两年最火的前端框架之一，组件化和虚拟DOM是其框架的核心思想。Flux则是react开发大型web项目时候经常用到的架构，单向的数据流动，与传统的MVC和MVVM等模式确实更加新颖。
1.react快速入门 http://www.ruanyifeng.com/blog/2015/03/react.html
2.react官方文档 http://reactjs.cn/react/docs/getting-started-zh-CN.html
3.flux入门教程 https://hulufei.gitbooks.io/react-tutorial/content/flux.html
4.flux阮一峰入门教程 http://www.ruanyifeng.com/blog/2016/01/flux.html

此外，还有babel（es6语法编译器）、scss（css预处理语言）、typings（js迁移到ts必用的工具）、fabricjs（基于canvas的开源画图工具）等一系列需要掌握的前端技术。真心感觉到前端的学无止境，上面这些技术很多都不是短时间内能熟练掌握的，自己作为初学者还需要不断学习！
