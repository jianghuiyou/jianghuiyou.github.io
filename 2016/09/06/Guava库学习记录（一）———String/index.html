Guava库学习记录（一）———String

  • Guava
  • Java SE

Guava是谷歌基于原生JDK进行封装而成的工具库，为了让java开发变得更加优雅高效，而Apache的commons库也是类似的工具库，不过在接口规范度和社区活跃度都略逊一筹。废话不多说，先从开发中最常用的String方向着手。

Joiner 字符串拼接工具

  • 可以设置跳过null选项，否则抛出异常。
  • 接收的参数可以是字符串数组，多个可变数量的字符串，迭代器接口的字符串。
  • 链式语法，Joiner对象是不可变的。

代码示例

Splitter 字符串分割工具

  • 灵活配置，功能强大。on方法调用可以按照单个字符、字符模式CharMatcher(后面会讲到)，正则表达式、字符串、固定长度进行分割，还可以对分割结果进行针对性的trim，例如裁剪掉空格、裁剪某类字符等等。
  • 返回结果可以是List（调用方法splitToList），也可以是Iterable（调用方法split）
  • 可以忽略结果集中的空选项（调用omitEmptyStrings方法）
    代码示例

CharMatcher 选定某种特种的字符集，并可进行操作

  • CharMatcher代表某类字符，比如空白字符、大写字符、数字字符、或者字符串中的任意一种（anyof(“aeiou”)代表元音字符）、某一种固定字符(is方法)。这是选定特征字符过程
  • 在选定之后通常要进行某些操作，例如上面提到的按此特征分割字符串（Splitter）、替换replaceFrom、保留retainFrom、裁边trimFrom、移除removeFrom等等。
    代码示例

此外，还有Charsets类内部有常见的编码格式，CaseFormat类可以变换不同的代码命名风格。由于并不常用此处不赘述。
