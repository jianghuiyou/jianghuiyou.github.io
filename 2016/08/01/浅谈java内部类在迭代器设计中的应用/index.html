浅谈java内部类在迭代器设计中的应用

  • java

通常我们认为Java内部类是作为一种代码隐藏机制存在，因为内部类对外部代码是不可知的（其实通过反射机制还是可知的）。但内部类的精髓远远不止这点，更加关键的是内部类与外围类的通信。

在这里，B就是内部类而A就称为外围类。

当通过一个外围类对象创建了一个内部类对象，其实内部类对象会自动拥有一个外围类对象的引用，并且可以直接访问外围类的所有成员，包括私有成员。
